# BMIcalculater
Docker Steps :

Step 1: Build an Image.

Step 2: Run the Container.

Pulling a Docker Image:

Go to the Docker Hub website [ https://hub.docker.com/ ]

Searching an images “hello world”.

You will find the command to run the image:

docker run hello-world

4. Copy the command and past in your terminal

Docker run –it ubuntu

Run the Container:

1. Now image convert into the container

2. Put this command in terminal:

docker run hello-world

3. Now stop docker running command:

docker stop <container ID>
